# Kickstart Scraper

## Description
app to collect information in ended fundraising to analyse 

## Setting 
```
pip install -r requirements.txt
```

## Run Local
```
python app.py
```

## Build to app
```
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name KickstarterScraper --icon=icon.con app.py
```
